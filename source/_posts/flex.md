#### flex布局是什么
Flex 是 Flexible Box 的缩写，意为"弹性布局"，用来为盒状模型提供最大的灵活性。
任何一个容器都可以指定为 Flex 布局。Webkit 内核的浏览器，必须加上-webkit前缀。
``` css
.box1{
  display:flex
  display:-webkit-flex
}
```
